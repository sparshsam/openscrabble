# Security Policy

## Supported Versions

OpenScrabble is a client-side application with no network backend. Security
vulnerabilities are limited to the application code itself.

| Version | Supported          |
|---------|--------------------|
| 0.3.x   | ✅ Active          |
| < 0.3   | ❌ Not supported   |

## Reporting a Vulnerability

OpenScrabble runs entirely in the browser with no data leaving the user's
machine. If you discover a security concern, please open a
[GitHub Issue](https://github.com/sparshsam/openscrabble/issues) rather than
a public discussion.

For sensitive reports, contact the maintainers directly via GitHub.
