## Getting Help

- **Issues** — report bugs or request features via [GitHub Issues](https://github.com/sparshsam/openscrabble/issues)
- **Discussions** — ask questions and share ideas via [GitHub Discussions](https://github.com/sparshsam/openscrabble/discussions)
- **Security** — report vulnerabilities via [Security Advisories](https://github.com/sparshsam/openscrabble/security/advisories)

## Before Opening an Issue

1. Check the [existing issues](https://github.com/sparshsam/openscrabble/issues) for duplicates.
2. Ensure you're on the latest version.
3. Clear your browser cache and try again.

## Feature Requests

Open an issue with the `enhancement` label. Describe what you'd like to see and why it would be useful.
