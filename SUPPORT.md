# Support

## Getting Help

OpenScrabble is a small, community-driven project. Support is best obtained
through the following channels:

### Documentation
- [README](README.md) — Quick start, gameplay, and controls
- [ARCHITECTURE](ARCHITECTURE.md) — Project structure and design

### Community
- [GitHub Issues](https://github.com/sparshsam/openscrabble/issues) — Bug
  reports, feature requests, and questions

## Supported Platforms

| Platform         | Support Status |
|------------------|----------------|
| Chrome / Edge    | ✅ Full        |
| Firefox          | ✅ Full        |
| Safari           | ✅ Full        |
| Mobile browsers  | ✅ Full        |

## Known Limitations

| Area          | Limitation |
|---------------|------------|
| Dictionary    | Built-in ~100K word list; not an official Scrabble dictionary |
| Networking    | No online multiplayer — local two-player only |
| AI            | No computer opponent — human vs human only |

## Unsupported Workflows

- Online multiplayer or matchmaking
- Accounts, profiles, or authentication
- Backend services or cloud sync
- AI or computer opponent
- Mobile app (responsive web only)
